# Student Management System (Java)

This is a simple Java console application that demonstrates Object-Oriented Programming concepts and file handling.

## Features

- Add a student
- List all students
- Search for a student
- Save student data to a file
- Load student data from a file

## Technologies Used

- Java
- ArrayList
- File Handling
- Object-Oriented Programming (OOP)

## Project Structure

StudentManagementSystem
│
├── Student.java
├── StudentManager.java
└── students.txt


## How to Run

1. Compile the files

javac Student.java StudentManager.java

2. Run the program

java StudentManager

## Learning Objectives

This project demonstrates:

- Encapsulation
- Classes and objects
- Collections (ArrayList)
- File storage
- Basic CRUD operations
